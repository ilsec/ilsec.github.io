+ (Program) new(options = {}) { ... }

Creates a new Assembly Program.

Examples:

ASM.new do
  mov  1, eax
  mov  1, ebx
  mov  2, ecx

  _loop do
    push  ecx
    imul  ebx, ecx
    pop   ebx

    inc eax
    cmp ebx, 10
    jl  :_loop
  end
end

    Additional options.

    The architecture of the Program.

  • :variables (Hash{Symbol => Object}) —

    Variables to set in the program.
