+ (Boolean) asm(path, options = {}) {|yasm| ... }

Assembles a file using yasm.

Examples:

Code.asm('code.s', :output => 'code.o')

Code.asm('file.s') do |yasm|
  yasm.output = 'code.o'
end

def Code.asm(path,options={},&block)

  return source_file.assemble(options,&block)
end
