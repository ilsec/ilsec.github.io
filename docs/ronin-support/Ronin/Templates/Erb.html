- (String) erb(template)

Renders the inline ERB template in the scope of the object.

Examples:

@user = 'lolcats'

erb %{
USER: <%= @user %>
PASSWORD: <%= @user.reverse %>
}
# => "\nUSER: lolcats\nPASSWORD: staclol\n"
