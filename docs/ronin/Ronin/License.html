Overview

Represents a software license and pre-defines many other common ones (License.bsd, cc<em>by, cc</em>by<em>sa, cc</em>by<em>nd, cc</em>by<em>nc, cc</em>by<em>nc</em>sa, cc<em>by</em>nc_nd, License.cc0, License.gpl2, License.gpl3, License.lgpl3 and License.mit).
