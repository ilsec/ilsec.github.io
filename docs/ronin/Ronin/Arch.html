Overview

Represents a Computer Architecture and predefines many other common architectures (Arch.x86, Arch.x86_64, Arch.ia64, Arch.ppc, Arch.ppc64, Arch.sparc, Arch.sparc64, Arch.mips and Arch.arm).

+ (Object) predefine(name, options = {})

This method is part of a private API. You should avoid using this method if possible, as it may be removed or be changed in the future.

Defines a new pre-defined Arch.

Examples:

Defining a pre-defined Arch

Arch.predefine :alpha, :endian => :big, :address_length => 8

Retrieving a pre-defined Arch

Arch.alpha

Parameters:

  • name (Symbol, String) —

    The name of the architecture.

  • options (Hash) (defaults to: {}) —

    Additional options.

Options Hash (options):

  • :endian (Symbol, String) —

    The endianness of the architecture.

  • :address_length (Integer) —

    The address-length of the architecture.
