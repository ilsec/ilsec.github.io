- (Style) initialize(options = {})

Sets the style options.

Parameters:

  • (Hash) options (defaults to: {}) —

    Style options.

Options Hash (options):

  • (Symbol) :case — default: DEFAULT_CASE —

    Controls the case of keywords. May be either :none, :lower, :upper or :random

  • (Symbol) :quotes — default: DEFAULT_QUOTES —

    Controls the quoting style of strings. May be either :single or :double.

  • (Boolean) :hex_escape — default: false —

    Forces all Strings to be hex-escaped.

  • (Symbol) :parens — default: DEFAULT_PARENS —

    Reduces the amount of parenthesis when tokenizing lists. May be either :less, :more.

  • (Boolean) :spaces — default: true —

    Controls whether spaces are used to separate keywords, or other kinds of white-space.

def initialize(options={})
  @case = (options[:case] || DEFAULT_CASE)
  @quotes = (options[:quotes] || DEFAULT_QUOTES)
  @hex_escape = (options[:hex_escape] || false)
  @parens = (options[:parens] || DEFAULT_PARENS)
  @spaces = true

  if options.has_key?(:spaces)
    @spaces = options[:spaces]
  end
end
