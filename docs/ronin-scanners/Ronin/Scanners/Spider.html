Overview

The Spider class represents a scanner that spider web pages, yielding Spidr::Page results and URL resources.

def agent(&block)
  options = {
    :proxy => Network::HTTP.proxy,
    :user_agent => Network::HTTP.user_agent
  }

  return Spidr::Agent.new(options,&block)
end
