+ (Object) test_for(name) {|page, callback| ... }

Adds a test to the Ronin::Scanners::WebVulnScanner. Tests should either return nil or a Vulns::WebVuln resource.

Examples:

Ronin::Scanners::WebVulnScanner.test do |page,callback|
  # ...
end

Parameters:

  • (Symbol) name —

    The name of the test.

Yields:

  • (page, callback) —

    The given block will be passed the scanner and the page currently being scanned.

Yield Parameters:

  • (Spidr::Page) page —

    The page currently being scanned.

  • (Proc) callback —

    The callback to pass back multiple Vulns::WebVuln objects.

def scan(&block)
  super do |page|
    WebVulnScanner.tests.each do |name,tests|
      tests.each { |test_block| test_block.call(page,block) }
    end
  end
end
