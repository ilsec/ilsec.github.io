- (Object) for_host(name_or_pattern) { ... }

Calls a given block for requests with matching Host headers.

Examples:

downloads = 0

get '/file' do
  for_host /^ftp/ do
    downloads += 1

    content_type :txt
    'some file'
  end

  for_host /^www/ do
    downloads += 1

    'some file'
  end

  for_host 'localhost' do
    "Total Downloads: #{downloads}"
  end
end

Parameters:

  • (Regexp, String) name_or_pattern —

    The exact name or pattern to match Host headers.

Yields:

  • [] The given block will be called when a request is received with the matching Host header.

def for_host(name_or_pattern)
  result = if name_or_pattern.kind_of?(Regexp)
             request.host =~ name_or_pattern
           else
             request.host == name_or_pattern.to_s
           end

  halt(*yield())
end
